# English Premier League Match Predictor

A machine learning project that predicts EPL match outcomes using team performance data.

## Features

- Predicts match results (Home Win, Draw, Away Win)
- Uses multiple ML models: MLP, Random Forest, Decision Tree
- Analyzes team league positions and performance

## Setup

1. Clone the repository
2. Create a virtual environment:
   python3 -m venv venv
   source venv/bin/activate
   3. Install dependencies:ash
   pip install pandas numpy scikit-learn scipy
   4. Add your data file to `data/2021-2022.csv`
5. Run the script:
   python3 main.pys
